Configures the attributes of widgets of the class "{}frame"{}. 

\begin{DoxyVerb}    Parameters obey the "optional parameters" protocol: if a parameter is "NULL", the state
    of the configured frame for that parameter does not change.
    When a frame is created, its parameters are set with the default values specified here.
\end{DoxyVerb}
 
\begin{DoxyParams}{Parameters}
{\em widget} & The widget to configure. \\
\hline
{\em requested\+\_\+size} & The size requested for this widget, including the widget\textquotesingle{}s borders. The geometry manager may override this size due to other constraints. Defaults to the "{}natural size"{} of the widget, ie. big enough to display the border and the text or the image. This may be (0, 0) if the widget has border\+\_\+width=0, and no text and no image. \\
\hline
{\em color} & The color of the background of the widget. Defaults to \doxylink{ei__types_8h_a00dfca48550b7224f2d8011497b9fd8c}{ei\+\_\+default\+\_\+background\+\_\+color}. \\
\hline
{\em border\+\_\+width} & The width in pixel of the border decoration of the widget. The final appearance depends on the "{}relief"{} parameter. Defaults to 0. \\
\hline
{\em relief} & Appearance of the border of the widget. Defaults to \doxylink{ei__types_8h_aa79a32b1d8ece0e44cfa394e870b270ba105222bc995071e84bce027c90a59b62}{ei\+\_\+relief\+\_\+none}. \\
\hline
{\em text} & The text to display in the widget, or NULL. Only one of the parameter "{}text"{} and "{}img"{} should be used (i.\+e. non-\/\+NULL). Defaults to NULL. \\
\hline
{\em text\+\_\+font} & The font used to display the text. Defaults to \doxylink{ei__types_8h_a722bfa1a8a9a4202fb2d4abce2f06a8c}{ei\+\_\+default\+\_\+font}. \\

{\em text\+\_\+anchor} & The anchor of the text, i.\+e. where it is placed within the widget. Defines both the anchoring point on the parent and on the widget. Defaults to \doxylink{ei__types_8h_a3852c963af609d31d7cfcff79c4c8450a5704b2469014b3e93fbdb0b297ed92b0}{ei\+\_\+anc\+\_\+center}. \\
