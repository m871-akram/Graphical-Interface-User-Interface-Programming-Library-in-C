Converts the red, green, blue and alpha components of a color into a 32 bits integer than can be written directly in the memory returned by hw_surface_get_buffer. The surface parameter provides the channel order.

Parameters
surfaceThe surface where to store this pixel, provides the channels order.
colorThe color to convert.
Returns
The 32 bit integer corresponding to the color. The alpha component of the color is ignored in the case of surfaces that don't have an alpha channel.
