Configures the attributes of widgets of the class "button".

Parameters
widget,requested_size,color,border_width,relief,text,text_font,text_color,text_anchor,img,img_rect,img_anchorSee the parameter definition of ei_frame_configure. The only difference is that relief defaults to ei_relief_raised and border_width defaults to k_default_button_border_width.
corner_radiusThe radius (in pixels) of the rounded corners of the button. 0 means straight corners. Defaults to k_default_button_corner_radius.
callbackThe callback function to call when the user clicks on the button. Defaults to NULL (no callback).
user_paramA programmer supplied parameter that will be passed to the callback when called. Defaults to NULL.
