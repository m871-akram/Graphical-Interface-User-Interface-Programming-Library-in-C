Creates a new instance of a widget of some particular class, as a descendant of an existing widget.

The widget is not displayed on screen until it is managed by a geometry manager. When no more needed, the widget must be released by calling ei_widget_destroy.

Parameters
class_nameThe name of the class of the widget that is to be created.
parentA pointer to the parent widget. Can not be NULL.
user_dataA pointer provided by the programmer for private use. May be NULL.
destructorA pointer to a function to call before destroying a widget structure. May be NULL.
Returns
The newly created widget, or NULL if there was an error.
